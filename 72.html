ಆತ್ಮಿಕ ಕಥೆಗಳು

ಸೈನಿಕನೂ ಅರಸನೂ

ನೆಪೋಲಿಯನ್ ಯುರೋಪನ್ನು ಆಳಿದ ಮಹಾ ಚಕ್ರವರ್ತಿಯಾಗಿದ್ದನು. ಅವನು ಬಡವರಿಗೆ ದಯೆತೋರಿಸುತ್ತಿದ್ದನು. ಆದರೆ ಅಹಂಕಾರಿಗಳನ್ನು ಅಸಹ್ಯಿಸಿದನು.

ಒಂದು ಸಾರಿ ನೆಪೋಲಿಯನ್ ತನ್ನ ಸೈನಿಕರ ಪಾಳ್ಯಕ್ಕೆ ಸಾಧಾರಣ ವಸ್ತ್ರಧಾರಿಯಾಗಿ ಹೋಗಿದ್ದನು. ಅಲ್ಲಿ ಒಬ್ಬ ಸೈನಿಕನು ತನ್ನ ಗುಡಾರದ ಹೊರಗೆ ಕೈಯಲ್ಲಿ ಚುಟ್ಟಾ ಹಿಡಿದುಕೊಂಡು ನಿಂತಿದ್ದನು. ಆಗ ನೆಪೋಲಿಯನ್ ಅವನ ಬಳಿಗೆ ಹೋಗಿ 'ಅಯ್ಯಾ, ಹಳ್ಳಿಗೆ ಹೋಗುವ ದಾರಿ ಯಾವುದು?' ಎಂದು ಕೇಳಿದನು.

ಆ ಸೈನಿಕನು ಅಹಂಕಾರಿಯಾಗಿದ್ದನು. ಅವನು ಚುಟ್ಟಾವನ್ನು ಬಾಯಲ್ಲಿಟ್ಟುಕೊಂಡು, ಆಕಾಶದ ಕಡೆ ನೋಡುತ್ತಿದ್ದನು. ನೆಪೋಲಿಯನ್ ಪುನಃ 'ಅಯ್ಯಾ, ಹಳ್ಳಿಗೆ ಹೋಗುವ ದಾರಿ ಯಾವುದು?'' ಎಂದು ಕೇಳಿದನು. ಆ ಸೈನಿಕನು ಕೋಪದಿಂದ, 'ಪೂರ್ವಕ್ಕೆ ಹೋಗು' ಎಂದು ಹೇಳಿ ರೈಲು ಗಾಡಿಯಂತೆ ಹೊಗೆ ಬಿಡಲಾರಂಭಿಸಿದನು.

ಆ ಸೈನಿಕನು ಗರ್ವಿಷ್ಟನೆಂದು ತಿಳಿದ ನೆಪೋಲಿಯನ್, ಅವನಿಗೆ ಬುದ್ಧಿ ಕಲಿಸಬೇಕೆಂದು ಅವನೊಡನೆ ಮಾತನಾಡಲಾರಂಭಿಸಿದನು. ನೆಪೋಲಿಯನ್ : ಅಯ್ಯಾ, ತಾವು ಯಾರೆಂದು ತಿಳಿದುಕೊಳ್ಳಬಹುದೇ? ಸೈನಿಕ : ಊಹಿಸಿ ತಿಳಿದುಕೋ ನೆಪೋಲಿಯನ್ : ತಾವು ಸೈನಿಕರೇ? ಸೈನಿಕ : ಇನ್ನು ಮೇಲಿನ ಸ್ಥಾನ! ನೆಪೋಲಿಯನ್ : ತಾವು ಲೆಪ್ಪಿನೆಂಟೋ? ಸೈನಿಕ : ಇನ್ನು ಮೇಲಿನ ಸ್ಥಾನ! ನೆಪೋಲಿಯನ್ : ತಾವು ಕೆಪ್ಪನೋ? ಸೈನಿಕ : (ಹೊಗೆ ಬಿಡುತ್ತಾ) ಇನ್ನೂ ಮೇಲೆ ನೆಪೋಲಿಯನ್ : ತಾವು ಮೇಜರೋ ? ಸೈನಿಕ : (ಹೆಮ್ಮೆಯಿಂದ ಬಾಯಲ್ಲಿ ಚುಟ್ಟಾವಿಟ್ಟುಕೊಂಡೇ) ಹೌದು. (ಸ್ವಲ್ಪ ಹೊತ್ತಿನ ಮೇಲೆ) ನನ್ನನ್ನು ಇಷ್ಟು ಪ್ರಶ್ನೆ ಕೇಳಿದ ನೀನು ಯಾರು? ನೆಪೋಲಿಯನ್ : ಊಹಿಸಿ ತಿಳಿದುಕೋ ಸೈನಿಕ : (ಅಲಕ್ಷದಿಂದ) ನೀನು ಸೈನಿಕನೋ ? ನೆಪೋಲಿಯನ್ : ಇನ್ನು ಮೇಲಿನ ಸ್ಥಾನ ಸೈನಿಕ : ನೀನು ಲೆಪ್ಟನೆಂಟೋ ? ನೆಪೋಲಿಯನ್ : ಇನ್ನೂ ಮೇಲೆ. ಸೈನಿಕ : ನೀನು ಕೆಪ್ಪನೋ ? ನೆಪೋಲಿಯನ್ : ಇನ್ನೂ ಮೇಲೆ. ಸೈನಿಕ : (ಬಾಯಿಂದ ಚುಟ್ಟಾ ತೆಗೆದು) ತಾವೂ ಮೇಜರೋ ? ನೆಪೋಲಿಯನ್ : ಇನ್ನೂ ಮೇಲೆ. ಸೈನಿಕ : (ಚುಟ್ಟಾ ಬಿಸಾಡಿಬಿಟ್ಟು ಭಯದಿಂದ) ತಾವು ನನ್ನ ಗೌರವದ ಜನರಲೋ? ನೆಪೋಲಿಯನ್ : ಇನ್ನೂ ಮೇಲೆ. ಸೈನಿಕ : (ಬಹು ಸಾಮ್ಯದಿಂದ) ತಾವು ಬಹು ಗೌರವದ ಪ್ರಧಾನ ದಳಪತಿಯವರೋ? ನೆಪೋಲಿಯನ್ : ಇನ್ನೂ ಮೇಲೆ.... ಸೈನಿಕ : (ನೆಪೋಲಿಯನ್ ಪಾದಕ್ಕೆ ಬಿದ್ದು) ತಾವುಗಳು ಮಹಾ.... ಮಹಾ.... ಚಕ್ರವರ್ತಿಯರಾದ... ನೆ...ಪೋ...ಲಿ..ನ್ ರವರೋ? ನೆಪೋಲಿಯನ್ : ಹೌದು. ಸೈನಿಕನು ಹೆದರಿಕೆಯಿಂದ ನಡುಗುತ್ತಾ, 'ನನ್ನ ಅಜ್ಞಾನವನ್ನು ಮನ್ನಿಸಬೇಕು." ಎಂದು ಕಣ್ಣೀರಿನಿಂದ ಬೇಡಿಕೊಂಡನು. ನೆಪೋಲಿಯನ್ ಅವನನ್ನು ಕ್ಷಮಿಸಿ ನಗುನಗುತ್ತಾ ಹೋದನು.

ತನ್ನನ್ನು ಹೆಚ್ಚಿಸಿಕೊಳ್ಳುವವನು ತಗ್ಗಿಸಲ್ಪಡುವನು. (ಮತ್ತಾಯ 23:2)
